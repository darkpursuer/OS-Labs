ATTENTION:
1. This code MUST be complied and run in Linux.

2. This code was written in C++.

3. You should use this command to compile this cpp file:
   >>  g++ Linker.cpp -o Linker -std=c++0x
   Then use this command to run:
   >> ./Linker - help

4. You can use the following command to handle with all the input files by given the directory path and then generate the output files in the specified output directory
   >> ./Linker -handle [inputdirpath] [outputdirpath]

5. To get the help of this program, you can use the following command:
   >> ./Linker -help
